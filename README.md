# psn-bibliothek

Local dev notes:

- Start server: `npm start` then open `http://localhost:8000`
- Update local games catalog (RAWG):

  - Optionally set `RAWG_KEY` environment variable.
  - Run: `npm run update-catalog`
  - This writes `data/games_catalog.json` used for autocomplete and images.
